Donkey Admin 백엔드 API 수정 요청해.

1. 새 API 추가: GET /admin/api/projects
   - Response: { "items": ["project-id-1", "project-id-2"] }
   - DB 요청 테이블에서 고유 project_id 목록 조회해서 반환

2. 기존 4개 API에 project_id 쿼리 파라미터(optional) 추가해서 필터링:
   - GET /admin/api/dashboard?project_id=xxx
   - GET /admin/api/usage?from_date=...&to_date=...&project_id=xxx
   - GET /admin/api/requests?page=1&limit=50&project_id=xxx
   - GET /admin/api/errors?period=week&project_id=xxx

project_id가 있으면 WHERE project_id = :project_id 로 필터링, 없으면 기존처럼 전체 조회.
요청 테이블에 project_id 컬럼은 이미 있어.
